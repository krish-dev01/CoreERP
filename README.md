# CoreERP 🚀

CoreERP is a **Java Spring Boot–based ERP system** designed to manage core business operations such as users, inventory, customers, orders, and invoices.  
This project is built step by step with **real-world backend architecture**, focusing on security, scalability, and clean code.

---

## 🔧 Tech Stack

- **Backend:** Java 17, Spring Boot 4
- **Security:** Spring Security (JWT – in progress)
- **Database:** MySQL
- **ORM:** JPA / Hibernate
- **Build Tool:** Maven (Maven Wrapper)
- **IDE:** VS Code / IntelliJ
- **API Style:** RESTful APIs
